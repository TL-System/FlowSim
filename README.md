flowsim

A.k.a. FlowSimulator
A.k.a. FlowsImproved

An generic event-driven flow-level network simulator for fast prototype implementation.
The goal is to evaluate rate control transport schemes for data center networks.

Topologies:
- FatTree
- SpineLeaf

Traffic arrival models:
- Poisson
- On/Off

Load balancing:
- ECMP
- Hedera
- Conga
